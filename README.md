# منصة R-Shop SaaS المتكاملة للتجارة الإلكترونية على نظام Android 🚀

أهلاً بك في الكود المصدري لمنصة **R-Shop SaaS** المخصصة للهواتف الذكية بنظام Android. تم بناء هذا التطبيق باستخدام **Kotlin** و **Jetpack Compose** كواجهة مستخدم متكاملة وحديثة مع **Room Database** لتخزين البيانات محلياً، وبوابة دفع ومحاكي رسائل واتساب متكاملين بالذكاء الاصطناعي (**Gemini API**).

---

## 📂 هيكل المجلدات والملفات الرئيسية (Project Architecture)

لقد قمنا ببناء هيكل متناسق ونظيف يفصل بين واجهات المستخدم، منطق العمل وقاعدة البيانات:

```text
/app/src/main/java/com/example/
│
├── MainActivity.kt                      # نقطة الدخول الرئيسية، تدفق الواجهات والتوجيه (Navigation Router)
│
├── data/
│   ├── model/
│   │   └── Entities.kt                  # جداول قاعدة البيانات: User, Store, Product, Order, Subscription
│   │
│   ├── dao/
│   │   └── AppDaos.kt                   # كائنات الوصول للبيانات (DAOs) لعمليات الاستعلام المتزامنة والتدفقات
│   │
│   ├── database/
│   │   └── AppDatabase.kt               # قاعدة البيانات الرئيسية (Room Database) وإعداداتها
│   │
│   ├── repository/
│   │   └── AppRepository.kt             # مستودع البيانات كوسيط مابين قاعدة البيانات والـ ViewModel
│   │
│   └── api/
│       └── GeminiHelper.kt              # دمج ذكاء اصطناعي تفاعلي مع نموذج Gemini 3.5 Flash للتوقعات والتنبؤ الذكي بالرسائل
│
└── ui/
    ├── screens/
    │   ├── UiComponents.kt              # بطاقات التصاميم الموحدة (M3)، حزم الألوان، ومربعات التنبيه المخصصة لواتساب و CCP
    │   └── SaaSAppScreens.kt            # الشاشات الكاملة: تسجيل الدخول، لوحة التاجر، إدارة المنتجات، الطلبات، ومصمم المتجر
    │
    └── theme/
        ├── Theme.kt                     # سمة التطبيق الأساسية (MyApplicationTheme) الداكنة المريحة للعين
        ├── Color.kt                     # لوحة الألوان الأساسية والثانوية للمنصة
        └── Type.kt                      # خطوط وأنماط النصوص
```

## 🛠️ كيف تبدأ بتشغيل المشروع وتثبيته (Getting Started)

1. **التحضير في أندرويد ستوديو (Android Studio)**:
   - افتح المجلد الرئيسي للمشروع في Android Studio.
   - سيقوم نظام Gradle بعمل تزامن (Sync) للملفات تلقائياً.

2. **ضبط مفتاح الذكاء الاصطناعي (Gemini API Key)**:
   - افتح لوحة الأسرار (Secrets Panel) في AI Studio وضع مفتاحك الخاص في حقل `GEMINI_API_KEY`.
   - أو قم بإنشاء ملف `.env` في المجلد الرئيسي واكتب:
     ```env
     GEMINI_API_KEY=your_real_gemini_api_key_here
     ```

3. **التشغيل والمحاكاة**:
   - قم بالضغط على **Run App** لتثبيته وتشغيله فوراً على المحاكي أو جهازك الحقيقي.
   - يدعم التطبيق التحديث المباشر للمخزن المحدث وإشعارات التنبيهات.
